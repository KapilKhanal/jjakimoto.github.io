Movie Review with Vector Representations

Date Fri, 13/01/2017 By Tomoaki Fujii Category Machine Learning. Tags NLP

Majority of data is expressed as text format such as news articles, SNS and speech. Each article has some meanings and opinions, which have significant effects on things around us like economics, politics, etc. Therefore, using these information for prediction systems is important. Most of prediction algorithms, however, take input in the form of tensor (1st-order tensor is vector), which requires us to represent text data in the tensor(vector) form.

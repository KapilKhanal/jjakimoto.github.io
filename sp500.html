Breaking Down S&P500;¶

S&P500; is compuated by linear combination of 500 stocks prices and weights are determined by market capitals. We can express as $$y = \bf{w}\cdot \bf{x} ,$$ where $y$ and $\bf{x}$ are S&P500; and stock prices; $\bf{w}$ and $\bf{x}$ are 500 dimensional vectors.

Hence, if $\bf{w}$ does not change over observations, S&P; 500 can be duplicated by vanila linear regression with stock prices.

We use three years data for training and predict an half an year S&P500.;

def get_pcaparm(pca, X, y, n_search=100):
    errors = []
    dim = X.shape[1]
    step = dim // n_search
    if step == 0:
        step = 1
    n_components_ = np.arange(1, dim, step)
    n_components = []
    kf = KFold(n_splits=5)
    for n in n_components_:
        errors_ = []
        for train, test in kf.split(X):
            try:
                prediction = pca_predict(pca, n, X[train], y[train], X[test], y[test])
                errors_.append(error(prediction, y[test]))
            except:
                continue
        if len(errors_) >= 1:
            errors.append(np.mean(errors_))
            n_components.append(n)
    return np.array(n_components)[np.argmin(errors)]

def pca_predict(pca, n, train_X, train_y, test_X, test_y):
    regr = LinearRegression()
    test_y = test_y.reshape([-1, 1])
    train_y = train_y.reshape([-1, 1])
    pca.n_components=n
    X_ = pca.fit_transform(train_X)
    regr.fit(X_, train_y)
    X_ = pca.transform(test_X)
    prediction = regr.predict(X_)
    return prediction

Prediction through Subsets of Stocks¶

In practical situation, you may use fewer stocks to constitute your portfoilo. Since S&P; is often used as bentch mark for performance of portforlios, we have to keep larger return and lower risk compared to S&P500.; Therefore, to hedge risk and maximize return, it is important to figure the relation between S&P500; and subset of stocks.

Automatic Relevance Determination¶

As we disucussed above, the contribution of each stock toward S&P500; will be detemined by the scale of company: i.e. market cap. Thus, only a few stocks may have dominant effect on S&P500.; In that case, focusing on such terms will make us avoid overfitting and have better generalization error. We introduce ARD(Automatic Relevance Determination).

Neural Network¶

If stocks are correlated each other only through linear relation, linear model is good enough to duplicate S&P500.; Especially, in the case stock data are all of the information, we can exactly solve stock data. As you see in the previous analysis, that is not the case. In this section, we will apply neural network to this task. Neural Network has been recently attracting a lot of attentions due to the great sucess in various fields like image recognition. According to universal approximation theorem, we can approximate almost any functions. More precisely, let $f$ be a borel function mapping from $\mathbb{R}^n$to $\mathbb{R}^r$. Let $\bf{X}$ a space of one layer simple feed forward neural network with a certain class activation function like sigmoid, tanh, Relu, etc. For any $\epsilon > 0$ and compact subset $K \subset \mathbb{R}^n$, there exists $\tilde{f} \in \bf{X}$ such that $$ \underset{x \in K}{sup}\| f(x) - \tilde{f}(x)\| < \epsilon.$$ In the original paper, more general statement has been shown. If you will not go into pure mathematics, you hardly see any nonborel functions. Hence, we can say that we can choose neural network for any funciton with any accuracy.

Note that this statement, however, is just quilitative. How to find such functions is another problem.

def pca_predict(n, train_input, train_target, test_input, model="pca"):
    if model is "pca":
        pca = PCA()
    elif model is "fa":
        pca = FactorAnalysis()
    regr = LinearRegression()
    Y_test = test_target.values.reshape([-1, 1])
    Y_train = train_target.values.reshape([-1, 1])
    pca.n_components=n
    X = pca.fit_transform(train_input.values)
    regr.fit(X, Y_train)
    X = pca.transform(test_input.values)
    prediction = regr.predict(X)
    return prediction

def get_param(train_input, target_input, valid_input, valid_target, model="pca", n_search=100):
    scores = []
    dim = train_input.values.shape[1]
    step = dim // n_search
    if step == 0:
        step = 1
    n_components = np.arange(1, dim, step)
    Y = valid_target.values.reshape([-1, 1])
    for n in n_components:
        prediction = pca_predict(n, train_input, train_target, valid_input, model)
        sc = testscore(prediction, Y)
        scores.append(sc)
    return n_components[np.argmin(scores)]
